### Week 4 Achievements:

* Selected the project topic: **A Centralized Registration and Log Collection System.**

* Identified key services related to the project:
  * **Amazon CloudWatch:** Monitoring and management service used to track logs, collect metrics, and set up automated system alerts.
  * **Amazon Simple Queue Service (SQS):** Message queuing service ensuring stable and reliable data transmission.
  * **Amazon Simple Notification Service (SNS):** Push notification service used for sending messages, emails, or triggering Lambda functions upon system events.
  * **AWS Lambda:** Serverless computing service that runs code in response to events without managing servers.
  * **Amazon Simple Storage Service (S3):** Object storage service providing highly durable and scalable storage for files and static data.
  * **Amazon DynamoDB:** NoSQL database service providing high performance and low latency at any scale.
  * **Amazon Athena:** Interactive query service that makes it easy to analyze data directly in S3 using standard SQL.
  * **Amazon Elastic Container Service (ECS):** Fully managed container orchestration service to run, stop, and manage Docker containers.
  * **Amazon Cognito:** Service for user registration, authentication, and access control.
  * **Amazon Kinesis Data Streams (KDS):** Real-time data streaming service for collecting and processing large streams of data records.
  * **Amazon Kinesis Data Firehose:** Reliable service to load streaming data into data lakes, data stores, and analytics services.

  > **Note:** However, the Kinesis services mentioned above are not available for Free Tier accounts, so they will not be used in the project.

* Mastered the process of deploying applications onto EC2 instances.
